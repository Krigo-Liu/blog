Definition - 2:

A subset U⊂M is open if for any x∈U, there eixsts some r>0 such that $$B_{r}(x)\subset U$$where Br={y∈M:d(y,x)<r} 概括：一个开集的子集中的任意一个点，与这个开集中的元素构成的ball，在这个子集里面。

Definition - 3:

A set of C⊂M is closed if xk∈C,limk→∞xk=x,x∈M which implies x∈C. 在closed set 里面找一个 subset，证明这个 subset 里面的数列的极限在这个subset里面，以为这在 closed set里

Equivalent:

A closed set contains all of its "limits": if x is the limit of some sequence xk∈C, then x∈C. 因此，我们发现 closed set 里面有其所有的数列的极限

Some facts - 2:

  • Intersection of closed sets is still closed. - closed 的交集是 closed
  • Let C1,C2 be closed in (M,d). Then C1∪C2 is closed. 同理，有限个 closed 的交集也是 closed 的
    • By induction, if C1,C2,...,Cn are closed, then ∪i=1nCi.
  • BUT!!! : Infinitely many closed sets, their union is not closed 无限个不是closed.
    • Example: Cn=[1n,2−1n] is closed for each n, but their union Cn=(0,3) is note closed.

Warning:

If a set is NOT open, then it is not true that it is closed, because set can be neither open nor closed
