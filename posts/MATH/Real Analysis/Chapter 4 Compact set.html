Theorem - 封闭youjie的嵌套序列

Any nested sequence of closed and bounded interval in R has nonempty intersection: if Ik=[ak,bk],such that a1≤a2≤...,b1≥b2≥b3... then $$\cap_{k=1}^{\infty}[a_{k},b_{k}]\neq\emptyset$$

Theorem: Heine-Borel

For a compact [a,b], for any collenction of open sets {Uα}α∈I "covering"$$[a,b],\cup_{\alpha\in I}U_{\alpha}\supset[a,b]$$ then there is a finite subset Ui such that $$\cup_{i=1}^{n}U_{i}\supset[a,b]$$

  • Example

From these properties of [a,b], we can define the notion of "compact set" in a metric space.

Definition of Compactness

Let (M,d) be a metric space. K⊂M.

Topological definition:

K is a compact set if any pen cover has a finite subcover.

  • Open cover: a collection of open sets {Uα}α∈I in M such that ∪α∈IUα⊃K.
  • Finite subcover: finitely many open sets {Uαi}i=1n in {Uα}α∈I such that it still covers K:∩i=1nUαi⊃K.

Sequential definition:

Any sequence {xk}k=1n in K has a convergent subsequence {xki}i=1∞ with limit also in K.

  • Subsequence: "1≥k1<k2<k3<..."

Theorem:

In any metric space, the 2 definitions above are equivalent: K is compact if and only if K is sequentially compact.

  • Compact sets are closed and bounded.

Continuous functions on Compact set

Opening example: Extreme Value Theorem

Let f:[a,b]→R be continuous. Then, there exists m,M∈R such that for all x∈[a,b],m≤f(x)≤M. Frthermore, there exists xmin,xmax∈[a,b] such that f(xmin)=m,f(xmax)=M. "Continuous functions on a closed and bounded interval attains its minmum and maximum values."

  • Step 1: show that a continuous function on [a,b] takes bounded value.
  • Step 2: inf/sup function values can be attained.
  • Proof Let K be a compact subset in a metric space X. Let Y be another metric space.
  • Recall two equivalent definitions of compactness:
    1. Any open cover has a finite subcover.
    2. Any sequence has a convergent subsequence

If K⊂X is compact and if f:X→Y is continuous, then the image f(K)⊂Y is compact.
