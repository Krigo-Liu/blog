Definition - 1:

f:X→Y is continuous at x0∈X if: $$\forall \epsilon>0,\exists \delta>0,d_{X}(x,x_{0})<\delta \Rightarrow d_{Y}(f(x),f(x_{0}))<\epsilon$$

Equivalent:

By using open ball:$$\forall \epsilon>0,\exists\delta>0,\forall x\in B_{\delta}(x_{0}),f(x)\in B_{\epsilon}(f(x_{0}))$$where Bδ(x0)={y∈X,d(y,x0)}⊂X and Bϵ(f(x0))={f(y)∈Y,d(f(y),f(x0))<ϵ}

Definition - 2:

f:X→Y is continuous on A⊂X if it is continuous at every x0∈A.

Theorem: if X,Y are metric spaces,

  1. f:X→Y is continuous at x0:$$\forall \epsilon>0,\exists \delta>0,d_{X}(x,x_{0})<\delta \Rightarrow d_{Y}(f(x),f(x_{0}))<\epsilon$$
  2. If xk→x0 in (X,dX), then f(xk)→f(x0) in (Y,dY)
  3. For all open set V⊂Y containing f(x0), there exist an open set U⊂X containing x0, such that f(U)⊂Y.

Some facts about continuous functions from Rm to Rn

Step 1:

Continuous functions f:Rm→R. Suppose f,g are both real-valued functions on Rm, and c∈R, then the following are also continuous functions:

  • Constant function c, and linear functions f(x1,...xm)=a1x1+...+amxm=aTx
  • f±g,cf,fg,fg; whenever g(x)≠0.
    • Proof idea: sequential criterion + algebraic limit theorem.
    • Therefore, polynomials in m variables and rational functions are continuous on their domain.
  • f, composed with continuous functions h:R→R
    • Examples:
    • max(f,g)=12(|f+g|+|f−g|),min(f,g)=12(|f+g|−|f−g|) are continuous

Step 2

Continuous functions f:Rm→Rn

  • f=(f1,f2,...,fn) where for each i=1,2,3,...,n, component fi:Rm→R
  • f is continuous if and only if each fi is continuous
    • Reason: in Rn with any p-norm, sequence converges iff each coordinate converges; combine this with the sequencetial criterion for continuity.
